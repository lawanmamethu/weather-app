A beautiful, real-time weather application that fetches live weather data from the **OpenWeather API**. Built with pure HTML, CSS, and Vanilla JavaScript.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Search any city** | Get instant weather updates for any city worldwide |
| 🌡️ **Real-time temperature** | Displays in Celsius and Fahrenheit |
| 💧 **Humidity & Pressure** | Complete weather details at a glance |
| 🌬️ **Wind speed** | Real-time wind data |
| 🔥 **Feels like temperature** | Perceived temperature based on humidity |
| ⏰ **Live clock & date** | Automatically updates in real-time |
| 🎨 **Stunning UI** | Sunset gradient with smooth animations |
| ⚡ **Optimized search** | Debounced input (500ms delay) prevents excessive API calls |
| 📱 **Fully responsive** | Works perfectly on mobile, tablet, and desktop |
| ⏳ **Loading states** | Visual feedback while fetching data |
| ❌ **Error handling** | User-friendly error messages for invalid cities |

## 🛠️ Technologies Used

| Technology | Purpose |
